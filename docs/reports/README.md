# Investigation and formulation reports

This directory holds historical investigation summaries and formulation analysis reports that were produced during development and verification of the element library and beam formulations. They are kept for reference and auditability.

- **ELEMENT_FORMULATION_*** — Analysis and fixes for element formulation (Euler-Bernoulli, Timoshenko, Levinson)
- **FINAL_INVESTIGATION_REPORT.md**, **INVESTIGATION_DIAGNOSTIC_REPORT.md** — General investigation notes
- **SHAPE_FUNCTION_FIX_SUMMARY.md** — Shape function property fixes
- **TIMOSHENKO_LEVINSON_ISSUES_FOUND.md**, **verify_timoshenko_levinson_formulations.md** — Timoshenko/Levinson verification
- **TIP_DEFLECTION_INVESTIGATION_SUMMARY.md** — Tip deflection investigations
- **VERIFICATION_SUMMARY.md** — Overall verification summary
