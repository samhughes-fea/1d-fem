# Euler–Bernoulli beam with Vlasov warping (7 DOF/node)

`LinearWarpingEulerBernoulliBeamElement3D` extends the classical 12-DOF EB element with warping intensity DOFs per node. Strain–displacement operators and linear EB material stiffness on the standard block live in [`../euler_bernoulli/utilities/`](../euler_bernoulli/utilities/).
